## 颜色方案

### 统计卡片

- **总账号数**: 靛蓝色图标 (indigo-600)
- **可用账号**: 绿色图标 (green-600)
- **禁用账号**: 红色图标 (red-600)
- **验证码密钥**: 
  - 已配置: 绿色图标 (green-600)
  - 未配置: 黄色图标 (yellow-600)

### 状态徽章

- **可用**: 绿色徽章 (bg-green-100 text-green-800)
- **禁用**: 红色徽章 (bg-red-100 text-red-800)

### 按钮

- **立即维护**: 靛蓝色 (bg-indigo-600 hover:bg-indigo-700)
- **刷新数据**: 白色边框 (border-gray-300 bg-white hover:bg-gray-50)

## 响应式布局

### 移动端 (< 768px)

- 统计卡片: 1 列
- 维护信息和快速操作: 1 列（垂直堆叠）
- 表格: 横向滚动

### 平板 (768px - 1024px)

- 统计卡片: 2 列
- 维护信息和快速操作: 2 列（并排）
- 表格: 正常显示

### 桌面 (> 1024px)

- 统计卡片: 4 列
- 维护信息和快速操作: 2 列（并排）
- 表格: 正常显示

## 动画效果

1. **加载动画**: 旋转图标 (animate-spin)
2. **悬停效果**: 按钮和卡片的阴影变化
3. **过渡效果**: 颜色和背景的平滑过渡

## 无障碍性

- 使用语义化 HTML 标签
- 提供 ARIA 标签
- 键盘导航支持
- 屏幕阅读器友好

## 图标使用

所有图标使用 Heroicons (Tailwind CSS 官方图标库):

- 用户组图标: 总账号数
- 勾选圆圈: 可用账号
- 叉号圆圈: 禁用账号
- 钥匙图标: 验证码密钥
- 刷新图标: 维护和刷新按钮
- 空盒子图标: 空状态

## 字体和间距

- 标题: text-3xl font-bold
- 卡片标题: text-lg font-semibold
- 统计数字: text-2xl font-bold
- 表格文本: text-sm
- 间距: 使用 Tailwind 的间距系统 (p-4, p-6, gap-6 等)
